Unit 1: HTML

In this unit we learned a little bit about the history of the Internet. We began making basic web pages that have the necessary framework elements such as <!DOCTYPE>, <html>, <head>, <title> and <body>. Next we learned about basic text formatting elements like headings and lists. We also learned to create hyperlinks and mark up advanced text such as blockquotes, pieces of code, dates and time, and much more. We learned about the importance of setting up the document structure correctly so assistive technologies like screenreaders can recognise those elements and help with tasks like "find the main navigation", or "find the main content." The last and most fun lesson in this unit was inserting images and video into our web pages.
